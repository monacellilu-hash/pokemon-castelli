# 💬 DIALOGHI NPC — Filler & Flavor
### Gente a caso per le città (battute random sui Pokémon + colore locale)

> Questi sono gli NPC "di riempimento": passanti che dicono cose buffe, danno mezzi
> consigli, o sparano cazzate a tema Castelli. Servono a far sembrare le città vive.
> In fase F9 diventano array di NPC in `data.js` (nome + righe di dialogo).
> Stile: romanesco leggero, logica del mondo Pokémon. Battute brevi.

## ➕ NOTE PER L'USO
- Mescolare gli NPC: alcuni danno **mezzi indizi** (meteorologo, vecchio del ponte, barcaiolo,
  custode museo), altri sono **puro colore** (porchetta, fragoline, traffico).
- Diversi NPC **anticipano la trama** senza spoilerare (grigioverde = GdF; "cantina strana" =
  Bunkerino/CoTrAL; "cosa grossa nel lago" = Kyogre).
- In F9 ognuno diventa un marker/voce con `mostraDialogo(nome, righe)`.
